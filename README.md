## N 体運動（非公式）

- N 体運動のシミュレーションを行う Fortran プログラムです。

- 2tai: 2体問題のシミュレーションを行います。
- 3tai: 3体問題のシミュレーションを行います。
- 3tai_dash: 3体問題のシミュレーションを行いますが、初期条件をわずかに変更しました。
- break_law: 逆二乗則を崩して、「逆3/4乗則」を適用したシミュレーションを行います。

- 3tai では、

    `u = [0.0d0, -0.2d0, 0.0d0]`

    という初期条件を用いていますが、これを3tai_dash では

    `u = [0.0d0, -0.205d0, 0.0d0]`

    とわずかに変更しています。

- いずれも、2次精度シンプレクティック法を用いています。
